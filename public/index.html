üîó API Endpoints

GET /auth

Description: Get Google OAuth authorization URL

Response: { "authUrl": "https://..." }

POST /auth/callback

Description: Handle OAuth callback with authorization code

Body: { "code": "authorization_code" }

Response: { "success": true, "message": "Authentication successful" }

GET /emails/senders

Description: Get emails grouped by sender (requires authentication)

Response: Array of sender objects with email counts and message IDs

POST /emails/delete

Description: Delete specified emails (requires authentication)

Body: { "messageIds": ["id1", "id2", ...] }

Response: { "success": true, "message": "Successfully deleted X emails" }
